# 자동차 도장 공정 MES Windows APP

팩토리 내부에서 수집된 센서 정보를 실시간 모니터링하고 장비를 제어하기 위한 WPF Application입니다. 

## 주요 기능

1. 센서 실시간 모니터링 기능
2. 임계치 알람 확인 기능
3. 색상 품질 검사 기능
4. 데이터 조회, 다운로드 가능 기능
5. PLC 제어 기능

## 실행 방법

*Visual Studio 기반 프로젝트입니다.*
.NET 8.0 버전 필요.
